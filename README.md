## 🚀 Featured Project — AI Portfolio Analyzer

🔗 **Live Demo:** https://portfolio-analyzer-sigma-amber.vercel.app/  
💻 **Source Code:** https://github.com/abhinavsharma11pix/portfolio-analyzer

An AI-powered portfolio analytics platform built for NSE and NYSE investors, combining quantitative finance, machine learning, and LLM-powered investment insights.

### Highlights

- 📈 Real-time market data with WebSocket streaming
- 🤖 AI portfolio advisor powered by Llama 3
- 📊 Risk analytics (Sharpe Ratio, VaR, Sortino, Beta, Max Drawdown)
- 🔮 30-day price forecasting using ensemble ML models
- 💰 Tax optimization with FIFO-based capital gains calculations
- 📄 Automated portfolio reports with AI-generated insights

**Tech Stack**
